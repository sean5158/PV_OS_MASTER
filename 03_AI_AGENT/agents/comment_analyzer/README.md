# Comment Analyzer Agent V1.0

> PV_OS 光伏客户评论分析 Agent
>
> 用于识别公开平台评论中的潜在光伏客户

---

# 一、Agent定位

名称：

Comment Analyzer Agent

版本：

V1.0


目标：

通过AI分析用户评论内容，识别：

- 家庭光伏需求
- 别墅光伏需求
- 小商业光伏需求
- 购买意向


---

# 二、输入数据

来源：

02_DATA/raw/


数据类型：

- 抖音评论
- 小红书评论
- 快手评论
- 视频号评论


标准格式：

02_DATA/data_dict/comment_schema.md


---

# 三、分析任务


## 1. 评论分类

判断：

- 是否与光伏相关
- 是否存在购买需求


## 2. 客户类型识别

分类：

- 家庭用户
- 别墅用户
- 小商业用户
- 同行业用户
- 无关用户


## 3. 购买意向判断


等级：

0 无需求

1 潜在兴趣

2 咨询意向

3 明确购买


## 4. 客户评分


输出：

0-100分


---

# 四、输出


输出位置：

05_CUSTOMER_CRM/leads/


包含：

- 用户标签
- 意向等级
- 客户类型
- 评分
- 原始来源
