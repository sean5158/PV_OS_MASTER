# 三、已经完成模块


## 1. 评论分析系统

文件：

02_DATA/04_COMMENT_DATABASE/

COMMENT_ANALYZER_RULE.md


功能：

- S/A/B/C评论分类
- 需求识别
- 用户类型识别
- 房屋场景识别


--------------------------------------------------


## 2. 评论时间资产系统


文件：

COMMENT_TIME_AND_MATCH_RULE.md


核心原则：

不是只采集最近7天。

规则：

最近7天：

销售重点窗口。


7天以上：

进入历史分析资产库。


用途：

- 关键词优化
- 用户画像
- 城市需求分析
- 竞品研究



--------------------------------------------------


## 3. 区域识别系统


规则：

优先级：

1 评论文本地点

2 IP属地

3 用户昵称

4 账号资料


支持：

四川

重庆

贵州


增加：

区域模糊识别。



--------------------------------------------------


## 4. 农村光伏客户规则


核心原则：

农村 ≠ 低价值。


农村客户分层：

A：
农村自费安装客户

B：
农村兴趣客户

C：
免费政策期待客户


判断：

购买意愿

>

房屋条件

>

区域

>

农村标签


免费安装：

降低评分。


但：

农村自建房+报价+安装需求

进入正常销售流程。



--------------------------------------------------


## 5. 客户评分模型

等级：

B 35-59
