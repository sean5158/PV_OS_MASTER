# 一、系统定位

AI Content Intelligence System 是 PV_OS 的内容生产与优化中心。

负责：

- 内容策略制定
- AI脚本生成
- 内容生产管理
- 平台发布
- 数据分析优化

# 三、核心模块


## 1. Strategy

路径：

strategy/


功能：

生成：

- 内容方向
- 用户痛点
- 选题规划
- 营销策略



---

## 2. Scripts AI

路径：

scripts_ai/


功能：

AI生成：

- 短视频脚本
- 图文文案
- 销售内容



---

## 3. Generation

路径：

generation/


功能：

管理：

- 视频素材
- 图片素材
- AI生成内容



---

## 4. Publishing

路径：

publishing/


功能：

管理：

- 抖音发布
- 小红书发布
- 快手发布
- 视频号发布



---

## 5. Analytics

路径：

analytics/


功能：

分析：

- 播放数据
- 用户反馈
- 转化效果
- 内容优化
