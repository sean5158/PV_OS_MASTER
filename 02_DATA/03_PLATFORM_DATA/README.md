# 03_PLATFORM_DATA —— 平台原始数据

## 目录用途

存放从抖音、小红书、快手、视频号采集的原始数据，是 PV_OS 数据管道的入口。数据清洗前的第一落点，保持采集时的原始形态。

典型内容：
- 视频 / 笔记元数据（标题、发布时间、互动量）
- 评论区原始内容（评论文本、用户标识、时间戳）
- 账号公开信息（昵称、简介、粉丝数）

## 数据类型

- JSON 格式的 API 原始响应或采集结果
- CSV 格式的结构化原始数据
- 按平台分目录存储：douyin/、xiaohongshu/、kuaishou/、shipinhao/

## 未来 AI 读取方式

数据清洗管道从此目录读取原始数据，处理后输出到 04_COMMENT_DATABASE 或 02_COMPETITOR_DATABASE。AI Agent 不直接读取原始数据，统一通过清洗管道消费。
