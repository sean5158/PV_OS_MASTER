# 06_SCORE_MODEL —— 评分模型数据

## 目录用途

存放客户评分模型的参数、规则、标注数据和评估结果。支撑 Customer Scorer Agent 的评分逻辑迭代优化。

典型内容：
- 评分规则定义（评分维度、权重、阈值）
- 标注数据集（已标注客户意图和价值的训练样本）
- 模型评估结果（准确率、召回率、误判分析）
- 评分规则版本历史

## 数据类型

- YAML / JSON 格式的评分规则配置文件
- CSV 格式的标注数据集
- Markdown 格式的模型评估报告
- 按版本分目录管理：v1/、v2/

## 未来 AI 读取方式

Customer Scorer Agent 启动时加载当前版本的评分规则配置，标注数据供模型迭代时使用。AI 按"当前版本 → 规则配置 → 评分执行"的三步读取流程。
