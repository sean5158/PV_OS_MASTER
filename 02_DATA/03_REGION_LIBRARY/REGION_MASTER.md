# REGION_MASTER —— PV_OS核心市场行政区域主数据 V1.0

> PV_OS 客户发现、关键词搜索、竞品发现、评分模型统一区域基础库。
>
> 核心目标：
> 以川渝黔为核心市场，建立省 → 城市 → 区县三级区域结构。
>
> 优先级：
> P0 成都
> P1 四川其他城市 + 重庆
> P2 贵州
