# COMMENT_REGION_RULE
# 评论区域识别规则 V1.0


版本:

V1.0


项目:

PV_OS_MASTER


目标:

从全国光伏相关评论中，

识别四川、重庆、贵州家庭光伏潜在客户。


核心原则:

全国采集。

区域筛选。

模糊识别。

不因地址不完整丢失客户。


---


# 一、区域识别目标


识别用户评论中的：

- 省级区域
- 城市区域
- 地域暗示


重点区域:


一级:

四川


二级:

重庆

贵州



---


# 二、识别来源优先级


|等级|来源|可信度|
|-|-|-|
|A|评论文字|最高|
|B|用户主动表达|高|
|C|IP属地|中|
|D|昵称/账号信息|低|



规则:

评论文字优先。

不要仅依靠IP判断。



---


# 三、模糊区域识别规则


## 3.1 省级表达


用户:

四川的


识别:


province:

四川


region_level:

B



---


## 3.2 城市表达


用户:

成都可以安装吗


识别:


province:

四川


city:

成都


region_level:

A



---


## 3.3 地区简称


例如:


川内可以装吗


识别:


province:

四川


region_level:

B



---


# 四、不完整地址保留规则


错误:

必须出现：

省+市+区

才算客户。


正确:


以下均保留:


我是四川的


成都多少钱


绵阳有没有安装


重庆可以做吗


贵州农村能装吗



---


# 五、未知区域处理


评论:


老家可以装吗


处理:


province:

unknown


region_level:

C


动作:

保留

进入二次AI判断
