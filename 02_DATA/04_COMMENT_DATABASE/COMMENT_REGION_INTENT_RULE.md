# COMMENT_REGION_INTENT_RULE
# 评论用户地域+购买意向联合识别规则 V1.0


## 一、目标

从全国光伏相关视频评论中：

识别：

- 四川潜在客户
- 重庆潜在客户
- 贵州潜在客户


原则：

竞品账号全国化。

客户地域本地化。


账号所在地不是客户判断依据。


---

# 二、地域识别原则


真实用户不会填写完整行政地址。


例如：

不会：

四川省成都市武侯区xxx


更多：

四川的可以装吗？

成都多少钱？

绵阳有没有安装？

重庆能做吗？


因此：

AI必须支持模糊地域识别。

# 四、客户意图联合判断


地域不是唯一条件。


必须结合需求。


高价值：

四川 + 多少钱

成都 + 怎么安装

重庆 + 有没有案例


判定：

销售线索。
