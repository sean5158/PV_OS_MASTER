# COMMENT_ANALYZER_RULE —— 评论区潜在客户识别算法 V1.1

> PV_OS 评论潜客发现系统核心规则。
>
> 目标：
> 从竞品评论区发现川渝黔家庭光伏潜在客户。
>
> 输出：
> - 高价值销售线索库
> - 潜客培育库
> - 评论分析数据库
>
> 核心原则：
> 不只寻找最近7天客户，而是建立长期评论资产库。
>
> 更新时间：2026-07-12


# 一、系统目标

PV_OS 从公开评论区中识别：

1. 有明确安装需求用户
2. 有兴趣但未决策用户
3. 有长期培育价值用户


评论来源：

- 抖音
- 快手
- 小红书
- 视频号
- 其他公开内容平台


分析对象：

- 竞品账号评论
- 光伏相关视频评论
- 家庭能源相关评论
