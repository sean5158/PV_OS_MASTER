# KEYWORD_STRATEGY —— 关键词发现与扩展策略 V1.0

> PV_OS 关键词全生命周期管理：从人工词根到 AI 扩展，从评分到竞品搜索投放。
>
> 最后更新：2026-07-11

## 一、关键词体系总览

### 1.1 关键词分类架构

```
PV_OS 关键词体系
├── 行业词       → 光伏、太阳能、储能、逆变器、组件、并网、离网
├── 场景词       → 别墅光伏、阳光房、露台改造、屋顶发电、民宿省电
├── 需求词       → 想装光伏、电费太高、光伏多少钱、补贴政策
├── 竞品词       → 竞品品牌名、竞品产品名、竞品创始人 IP
├── 区域词       → 成都光伏安装、重庆太阳能、贵阳光伏公司
└── 长尾词       → 多词组合的细粒度搜索词
```

### 1.2 关键词生命周期

```
词根输入 → 平台扩展 → 数据验证 → AI评分 → 分级入库 → 搜索投放 → 效果回收 → 迭代优化
```

## 二、人工词根输入机制

### 2.1 词根来源

| 来源 | 说明 | 示例 |
|------|------|------|
| 行业知识 | 光伏行业基础术语 | 光伏、组件、逆变器、储能 |
| 业务经验 | 一线销售/运营的实战关键词 | 屋顶漏水做光伏、阳光房隔热 |
| 客户原话 | 真实客户咨询中的自然表达 | 我家电费一个月八百能不能装光伏 |
| 政策驱动 | 新政策引发的搜索热点 | 2026光伏补贴、分时电价 |
| 竞品观察 | 从竞品内容中提取的高频词 | 竞品反复使用的标签/标题词 |

### 2.2 词根录入格式

```
词根文件：02_DATA/01_KEYWORD_LIBRARY/seed_keywords.yml

格式：
- keyword: "别墅光伏"
  category: "场景词"
  sub_category: "高端住宅"
  source: "人工"
  initial_score: 85
  region: ["四川", "重庆", "贵州"]
  notes: "高客单价场景，直接竞品搜索使用"
```

### 2.3 词根维护规则

- 词根库按**月度**审视更新，由人工触发
- 新增词根标记 `source: "人工"` 和录入日期
- 废弃词根不移除，标记 `status: "deprecated"` 保留追溯
- 词根数量控制在 50-100 条，保持精炼

## 三、平台搜索联想词获取

### 3.1 原理

利用各平台搜索框的 **Suggest API**，输入词根 → 获取平台自动补全的联想词列表，从而发现真实用户的搜索习惯。

### 3.2 各平台获取方式

| 平台 | 联想词特征 | 优先级 |
|------|-----------|:------:|
| **抖音** | 搜索联想词反映当下热点和用户真实搜索，更新极快 | 🔴 高 |
| **小红书** | 联想词偏消费决策和种草场景，长尾词丰富 | 🔴 高 |
| **快手** | 联想词偏下沉市场，口语化程度高 | 🟡 中 |
| **视频号** | 联想词偏社交传播和信任决策 | 🟡 中 |

### 3.3 扩展策略

```
词根："别墅光伏"
  │
  ├── 抖音联想 → 别墅光伏多少钱、别墅光伏划算吗、别墅光伏安装实拍
  ├── 小红书联想 → 别墅光伏设计、别墅光伏案例、别墅光伏避坑
  ├── 快手联想 → 别墅装光伏、别墅光伏发电、别墅光伏全套
  └── 视频号联想 → 别墅光伏推荐、别墅光伏靠谱吗、别墅光伏公司
```

### 3.4 联想词入库规则

- 每条联想词记录来源平台和获取时间
- 去重：同一关键词多平台出现视为高价值信号
- 联想词自动标记 `source: "platform_suggest"`

## 四、平台标签扩展

### 4.1 标签获取来源

| 来源 | 说明 |
|------|------|
| 爆款视频/笔记的标签 | 高互动内容使用的标签，反映流量密码 |
| 竞品常用标签 | 竞品反复使用的标签组合 |
| 平台热门标签 | 平台侧推流的 Trending Tag |
| 话题标签 | 官方话题活动的关联标签 |

### 4.2 标签扩展示例

```
竞品视频标签：#光伏 #别墅装修 #阳光房 #节能环保
        │
        ├── 提取 → 光伏、别墅装修、阳光房、节能环保
        ├── 关联扩展 → 别墅设计、屋顶改造、光伏阳光房、绿色建筑
```

### 4.3 标签入库

- 标签关键词存入 `tag_keywords.csv`
- 记录：标签文本、出现频次、来源账号、首次发现日期
- 高频标签（≥5 个竞品使用）自动提升评分

## 五、竞品标题关键词提取

### 5.1 提取目标

从已入库竞品（S/A/B 级）的作品标题中，提取高频关键词和标题结构模式。

### 5.2 提取维度

| 维度 | 说明 | 示例 |
|------|------|------|
| 高频实词 | 标题中频繁出现的名词/动词 | 光伏、安装、省钱、别墅 |
| 钩子词 | 标题前 15 字中的吸引点击词汇 | 惊呆了、千万别、终于 |
| 数字词 | 标题中的数字表达 | 3年回本、月省800、5kw |
| 情感词 | 情绪化表达 | 后悔没早装、太香了、真香 |
| 场景词 | 具体应用场景 | 阳光房、露台、屋顶 |

### 5.3 提取流程

```
竞品标题池（S/A级竞品最近90天标题）
  → 分词 + 去停用词
  → 词频统计（按出现次数降序）
  → 按五大维度分类
  → 与现有关键词库交叉比对
  → 未覆盖的高频词 → 作为候选补充词根
```

## 六、评论区用户自然语言关键词

### 6.1 价值定位

评论区是**用户真实需求的富矿**。用户在评论中使用的自然语言，比搜索关键词更贴近实际购买决策。

### 6.2 提取对象

| 评论来源 | 提取重点 |
|----------|---------|
| 竞品评论区 | 用户在竞品下的咨询和购买意向表达 |
| 行业爆款评论区 | 泛光伏受众的疑问和兴趣点 |
| 家居/装修类评论区 | 跨品类的高净值用户需求信号 |

### 6.3 自然语言 → 关键词映射

| 用户评论原话 | 提取关键词 | 语义归类 |
|-------------|-----------|---------|
| "我家是叠拼，能不能装光伏" | 叠拼光伏 | 场景词 |
| "装了光伏一个月能省多少电" | 光伏省电、光伏收益 | 需求词 |
| "成都这边有靠谱的安装公司吗" | 成都光伏安装 | 区域词+需求词 |
| "和邻居一起装会不会便宜点" | 光伏团购、光伏拼装 | 长尾词 |
| "阳光房太晒了想铺光伏板" | 阳光房光伏、光伏隔热 | 场景词+需求词 |

### 6.4 提取规则

- 评论中出现 **疑问句** → 高价值信号，优先提取
- 评论中包含 **地点信息** → 标记区域属性
- 评论中包含 **数字**（面积、金额、时间）→ 标记为精准需求
- 同类表达 ≥3 次 → 生成正式关键词入库

## 七、AI 关键词扩展

### 7.1 扩展机制

当词根库积累到一定规模后，由 AI 进行语义扩展，发现人工未覆盖的关联词。

### 7.2 扩展方向

| 扩展类型 | 说明 | 示例 |
|----------|------|------|
| **同义扩展** | 同一含义的不同表达 | 光伏安装 ≈ 光伏施工 ≈ 装光伏 ≈ 安光伏 |
| **上下位扩展** | 泛化或细化 | 别墅光伏 → 高端住宅光伏 → 独栋光伏 |
| **场景关联** | 相关场景的横向扩展 | 阳光房 → 露台 → 花园 → 屋顶平台 |
| **需求链扩展** | 决策链条上的前后环节 | 光伏价格 → 光伏补贴 → 光伏并网 → 光伏售后 |
| **区域组合** | 关键词 × 川渝黔城市 | 光伏安装 × [成都, 重庆, 贵阳, 绵阳...] |

### 7.3 扩展约束

- AI 扩展的关键词标记 `source: "ai_expand"` 
- 扩展结果需人工抽检确认（抽检率 ≥20%）
- 不与现有关键词库重复
- 每次扩展量控制在 30 条以内，保持质量

## 八、关键词评分规则

### 8.1 评分维度

| 维度 | 满分 | 权重 | 说明 |
|------|:----:|:----:|------|
| 业务匹配度 | 30 | 30% | 与光伏目标业务的直接相关性 |
| 转化潜力 | 30 | 30% | 该关键词背后的用户是否有购买意向 |
| 搜索热度 | 20 | 20% | 多平台出现的频率和广度 |
| 竞品覆盖度 | 10 | 10% | 该关键词是否被竞品大量使用 |
| 区域匹配 | 10 | 10% | 与川渝黔市场的关联度 |
| **总分** | **100** | **100%** | — |

### 8.2 评分细则

**业务匹配度（0-30）：**

| 分数 | 标准 |
|:----:|------|
| 25-30 | 直接指向光伏安装/购买决策（"光伏安装""光伏报价"） |
| 18-24 | 与光伏强相关但非直接购买（"光伏发电原理""光伏政策"） |
| 10-17 | 新能源泛领域或家居装修相关（"省电""屋顶改造"） |
| 0-9 | 弱关联或无关 |

**转化潜力（0-30）：**

| 分数 | 标准 |
|:----:|------|
| 25-30 | 含明确购买意图词（"多少钱""怎么装""推荐""靠谱吗"） |
| 18-24 | 含比较/评估意图（"划算吗""效果""对比"） |
| 10-17 | 含了解/学习意图（"是什么""原理""政策"） |
| 0-9 | 纯信息浏览意图 |

**搜索热度（0-20）：**

| 分数 | 标准 |
|:----:|------|
| 16-20 | 3+ 平台联想词均出现 |
| 11-15 | 2 个平台出现 |
| 6-10 | 单平台出现 |
| 0-5 | 仅在竞品标题/标签中出现 |

**竞品覆盖度（0-10）：**

| 分数 | 标准 |
|:----:|------|
| 8-10 | ≥5 个竞品账号内容中使用 |
| 5-7 | 2-4 个竞品使用 |
| 2-4 | 1 个竞品使用 |
| 0-1 | 未被竞品使用（蓝海信号或无效词） |

**区域匹配（0-10）：**

| 分数 | 标准 |
|:----:|------|
| 8-10 | 包含川渝黔明确地名 |
| 5-7 | 西南区域泛指向 |
| 2-4 | 无区域限定但从评论区推断区域属性 |
| 0-1 | 无区域关联 |

### 8.3 关键词等级划分

| 等级 | 总分 | 标签 | 用途 |
|:----:|:----:|------|------|
| **S** | ≥80 | 核心关键词 | 优先投放竞品搜索，每日监控 |
| **A** | 60-79 | 重要关键词 | 竞品搜索使用，周度观察 |
| **B** | 40-59 | 常规关键词 | 辅助搜索，月度审视 |
| **C** | <40 | 观察关键词 | 不入库，季度复查是否可升级 |

## 九、高价值关键词进入竞品搜索流程

### 9.1 触发条件

同时满足以下条件的关键词，自动进入竞品搜索队列：

- 关键词等级 ≥ **A 级**（总分 ≥60）
- 业务匹配度 ≥ **18 分**
- 转化潜力 ≥ **18 分**
- 未被竞品覆盖 或 被 ≥3 个竞品使用（两端信号均有价值）

### 9.2 投放策略

```
S 级关键词（≥80）
  → 每日搜索 1 次
  → 搜索结果前 30 条账号逐一评分
  → 新发现竞品直接进入 COMPETITOR_SCORE_RULE 判定

A 级关键词（60-79）
  → 每 3 天搜索 1 次
  → 搜索结果前 20 条中未入库账号纳入候选
  → 批量评分后择优入库

B 级关键词（40-59）
  → 每周搜索 1 次
  → 仅记录搜索结果变化，发现新面孔时触发评分
```

### 9.3 搜索组合策略

高价值关键词不是孤立使用，而是组合投放：

```
单关键词搜索：       "别墅光伏"
关键词组合搜索：     "别墅光伏" + "安装"
场景 + 区域组合：    "阳光房光伏" + "成都"
需求 + 场景组合：    "光伏多少钱" + "别墅"
竞品词 + 区域组合：  "[竞品品牌]" + "重庆"
```

组合规则：
- S 级关键词生成 3-5 组组合搜索
- A 级关键词生成 1-2 组组合搜索
- 区域组合默认覆盖成都、重庆、贵阳三城

### 9.4 搜索效果回收

每次搜索投放后，AI 记录以下指标用于关键词价值迭代：

| 指标 | 说明 |
|------|------|
| 新发现竞品数 | 该关键词组合发现的新账号数 |
| 入库转化率 | 新发现账号中达到入库标准的比例 |
| S/A 级占比 | 入库账号中 S/A 级比例 |
| 噪声率 | 搜索结果中与光伏无关的比例 |

连续 3 次搜索零产出的关键词 → 降级或移出搜索队列。
