# 01_KEYWORD_LIBRARY —— 关键词库

## 目录用途

存放光伏行业运营所需的全部关键词，支撑竞品账号发现和评论潜客识别的搜索与匹配。

典型内容：
- 光伏行业词（组件、逆变器、储能、并网、离网）
- 场景词（别墅光伏、阳光房、露台改造、民宿省电）
- 竞品品牌词（竞品公司名、产品名）
- 客户意图词（想装光伏、电费太高、光伏多少钱）
- 负面排除词（降低误匹配率）

## 数据类型

- Markdown / YAML 格式的关键词分类清单
- CSV 格式的关键词-权重映射表
- JSON 格式的关键词语义向量索引

## 未来 AI 读取方式

AI Agent 在执行竞品发现或评论分析任务时，首先从此目录加载关键词库作为搜索种子和匹配规则。读取优先级：先读分类索引文件，再按任务类型定向加载对应分类的关键词表。
