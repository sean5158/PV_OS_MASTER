# 一、账号分类


## TYPE_A

个人光伏内容博主


特点:

- 个人运营
- 高频视频输出
- 光伏内容占主体
- 评论区存在客户咨询


判断:

名称相关性
+
视频内容相关性
+
评论需求相关性


权重最高。


---


## TYPE_B

企业/安装公司账号


包括:

- 光伏安装公司
- EPC
- 品牌直营
- 厂家服务账号


判断:

公司属性
+
安装案例
+
服务区域


---


## TYPE_C

行业参考账号


包括:

- 新闻
- 科普
- 产品介绍
- 厂家技术账号


保存观察。
