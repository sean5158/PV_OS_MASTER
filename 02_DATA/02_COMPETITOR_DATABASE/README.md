# 02_COMPETITOR_DATABASE —— 竞品账号数据库

## 目录用途

存放已发现的竞品账号信息，是竞品发现系统的核心数据资产。支撑竞品自动发现 Agent 的增量更新和竞品分析 Agent 的深度分析。

典型内容：
- 竞品账号基础档案（平台、ID、粉丝量、作品量）
- 竞品内容发布记录（更新频率、选题方向）
- 竞品互动数据（点赞、评论、分享趋势）
- 竞品转化线索（如何引导私域、评论区转化话术）

## 数据类型

- CSV 格式的竞品账号主表（competitor_master.csv）
- JSON 格式的单账号详细档案
- Markdown 格式的竞品分析报告

## 未来 AI 读取方式

Competitor Discovery Agent 定期读取主表进行增量发现，Competitor Analyst Agent 读取账号详细档案进行深度分析。AI 按"先主表定位 → 再档案深入"的两级读取策略，避免全量加载。
