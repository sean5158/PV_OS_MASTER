## 一、定位

Content Intelligence 是 Inbound 闭环的前端引擎。

**输入**: video_asset_store.csv（竞品视频资产）
**输出**: 二创脚本、内容策略、选题方向

```
竞品视频资产
    ↓
AI爆款拆解
    ├── 为什么爆？
    ├── 黄金三秒是什么？
    ├── 用户痛点是什么？
    ├── 评论触发点是什么？
    ├── 转折在哪里？
    └── 成交因素是什么？
        ↓
    AI二创脚本
        ↓
    人工审核+拍摄
        ↓
    自有账号发布
```

## 二、爆款拆解维度

### 2.1 九维拆解模型

| 维度 | 字段 | 说明 |
|------|------|------|
| **钩子** | `hook_3_seconds` | 前三秒说了什么让人停下？ |
| **痛点** | `pain_point` | 击中用户什么焦虑/需求？ |
| **客群** | `customer_type` | 面向谁？别墅业主/小商家/普通家庭？ |
| **结构** | `video_structure` | 开头-中段-结尾的结构模式 |
| **标题** | `title_pattern` | 标题公式（数字型/疑问型/对比型） |
| **评论触发** | `comment_trigger` | 什么内容让观众忍不住评论？ |
| **爆款原因** | `viral_reason` | 综合判断：情绪/实用/反差/共鸣？ |
| **转折点** | `turning_point` | 视频在哪个点情绪/信息发生转折？ |
| **成交因素** | `closing_factor` | 哪些因素促使用户产生购买意向？ |

### 2.2 拆解输出格式

```json
{
  "video_id": "dy_v01234",
  "analysis": {
    "hook_3_seconds": "数字对比：电费从3000降到300",
    "pain_point": "城市别墅业主电费高但不知道光伏能省多少",
    "customer_type": "别墅业主",
    "video_structure": "痛点开场 → 方案展示 → 数据对比 → 客户见证 → 引导咨询",
    "title_pattern": "数字对比型：我家XX平米装了光伏，一年省了X万",
    "comment_trigger": "评论里很多人问'我家能装吗'——制造了代入感",
    "viral_reason": "强数字对比 + 真实案例 + 本地信任感",
    "turning_point": "第15秒展示电费账单对比",
    "closing_factor": "本地案例 + 可上门测量 = 降低决策门槛"
  },
  "reusable_elements": {
    "hook_template": "【数字】从【旧状态】变成【新状态】",
    "structure_template": "痛点→方案→数据→见证→CTA",
    "key_phrases": ["我家XX平米", "一年省了X万", "真实案例"]
  }
}
```

## 三、AI二创脚本生成

### 3.1 生成流程

```
Step 1: 选择拆解模板
    从 viral_analysis/ 选择 1-3 个高分爆款拆解
        ↓
Step 2: 差异化角度
    PV_OS 是四川IP博主 → 加入本地化元素
    不是模仿竞品 → 是借鉴结构 + 替换内容
        ↓
Step 3: AI生成脚本
    调用 ChatGPT/DeepSeek 根据拆解模板生成脚本
        ↓
Step 4: 人工审核
    检查: 真实性/专业度/本地化/差异化
        ↓
Step 5: 存入 scripts_ai/
    标记: 来源拆解/生成模型/审核状态
```

### 3.2 本地化差异原则

PV_OS = 四川IP博主。二创时必须差异化：

| 竞品内容 | PV_OS 差异化 |
|---------|------------|
| 北京别墅案例 | → 成都别墅案例 |
| 全国品牌宣传 | → 四川本地安装实拍 |
| 科普光伏原理 | → 算账：成都日照条件下能省多少 |
| 工厂生产线 | → 施工现场：成都某小区安装过程 |
| "全国服务" | → "成都/重庆/贵阳本地服务" |

### 3.3 脚本存储

```
04_CONTENT/scripts_ai/
├── 2026-07-20_别墅光伏避坑指南_v1.md
├── 2026-07-20_阳光房光伏省钱计算_v1.md
└── 2026-07-20_成都老旧小区光伏安装_v1.md

每个脚本包含:
- 来源拆解视频ID
- AI模型 + 版本
- 脚本正文 (分镜)
- 人工审核状态
- 发布状态
```

## 四、内容策略输出

### 4.1 周期输出

| 周期 | 输出 | 内容 |
|------|------|------|
| 每周 | `content_insight.json` 更新 | 本周热门选题、需求缺口、标题模式 |
| 每月 | `content_strategy.md` | 月度内容方向调整、竞品趋势 |
| 每季 | 爆款模式总结 | 季度爆款共性规律 |

### 4.2 与自有账号的连接

```
content_intelligence
        ↓
content_calendar (发布计划)
        ↓
自有账号发布
        ↓
content_performance (效果追踪)
        ↓
content_to_lead_mapping (线索归因)
        ↓
反馈回 content_intelligence (优化策略)
```
