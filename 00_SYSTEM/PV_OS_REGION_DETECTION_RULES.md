# PV_OS_REGION_DETECTION_RULES

版本：V1.0

日期：2026-07-19

用途：

定义 PV_OS 客户发现系统区域识别规则。

本文件是区域判断统一规则源。

所有 Agent、Workflow、评分模型不得自行定义区域规则。

---

# 一、区域识别目标

PV_OS目标市场：

四川

重庆

贵州


客户发现范围：

全国公开平台评论数据。

包括：

- 抖音
- 小红书
- 快手
- 视频号


AI需要从全国评论中发现：

- 四川客户
- 重庆客户
- 贵州客户

# 六、禁止规则


禁止：

- 只使用评论文本判断区域

- 只使用IP判断区域

- 因为没有城市名称删除客户
